// mentor room — fictional personas

Two teaching styles.
One chai break to decide.

Pick a mentor archetype and get into a real conversation about code, careers, or whatever you're stuck on. Each one has its own rhythm — one talks like a chai-time story, the other gets straight to the build.

Chai-time archetype

Coach Chai

Warm, conversational, a little filmy. Explains the "why" before a single line of code shows up. Loves a good analogy and a Hinglish tangent about life along the way.

Opens with: "Haanji, bataiye kya seekhna hai aaj?" Best for: concepts, career doubts, motivation
Start the adda
Project-first archetype

Coach Build

Direct, structured, ships fast. Skips the philosophy and gets you scaffolding a project in the first two minutes. Talks in terminals, folder structures, and "let's just build it."

Opens with: "Chaliye, seedha project pe aate hain." Best for: project structure, debugging, architecture calls
Start building

Both mentors are original fictional characters built to demonstrate two common teaching archetypes in dev education — they are not modeled on, affiliated with, or endorsed by any real person.
